File menu
file menu 


Open...
This item selects a file and tests whether the data are in some supported binary format or ascii format. A binary file is loaded into memory and written to the Editor; a text file is opened in the Editor. Thereafter the data is handled as any data put onto the clipboard.
Also the multiple-curve formats rfb, PTW-mcc and RFA300 are supported. With the keyboard the appropriate curve may by selected. This number is stored when the preferences are saved.
As alternative, a file may be dropped onto the graph area. Multiple files can be dropped or opened when the history list is activated.

Open Temporary Reference...
This item selects a file and stores it into memory as temporary reference. This is the equivalent of opening a file (as measurement) and Setting as Reference.  For unstructured multi reference files reference indexing needs to be activated.

Load current data
With this option the current data source (clipboard or file) is reread from the editor window with complete data-processing.


Save Measurement as...

When the editor tab is selected these data will be stored as plain text, otherwise the measurement curve can be stored to disk in the OmniPro_v6 format (most complete), PTW-mcc, wms-ascii, wms-binary and RFA-300 format.

Save Selected Curve as...
When the coloured name/date label is clicked, this will select the corresponding curve in the graph area will be selected.
Then this curve can be stored as decribed above. 

Ignore Clipboard
As default the program hooks up to the chain of clipboard viewers in the windows system and checks all data that are placed on the clipboard. This behavior may be switched off. When switched off the program itself puts analysis information on the clipboard in Special mode 2.
Loading a binary file, which is not supported through the clipboard, will lock the clipboard; loading a text-based file will restore the last know state for the clipboard. The user can change the clipboard state at any time.

Save as Reference

The current file, as found in the editor window, is saved as a reference under a specific name which is dependent on linac, energy, field size, wedge and type of scan. This generally covers all relevant parameters for a reference file as intended here.
This feature is disabled by default and is reset at program start. To activate this item use <Ctrl>+<Alt>+<R>. Locking may be done again in the same way.
When a muitiple curve file is read and Device specific Reference is not active a single curve OmniPro_v6 text file will be written.
The Reference directory is set at installation time but can be changed.

Use History list
When checked more data sets can be held in memory. See History topic and settings.

The printing facility is kindly provided by Alan Chamberlan.
Print Active page
This will generate a pdf of the current tab. Note that the fonts directory must be installed.

Print Selected tabs
This will generate a pdf of all selected tabs. Note that the fonts directory must be installed.
