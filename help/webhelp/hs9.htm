Below the bottom axis the orientation of the curve with respect to linac coordinates is denoted with AB / TG / UD in the order as defined by the axis direction in the data.

Zooming
Zooming of the vertical scale can be done with the <Up> and <Down> keys. The keys <Z> and <U> set or unset zoomed state. Graphical zooming can be done with the mouse left button pressed selecting an area from top-left to bottom-right. The other way around for unzoom.

Graphical elements
Besides the obvious data curves there are helper lines (Top model, FFF slopes, penumbras, residual error dots), indicators (In-Field Area) and cursors. All of these elements can be toggled on/off in the View menu or simple keyboard shortcuts. See also curve selection below.

Curve selection
Data curves can be selected by clicking on corresponding label. Pressing the <spacebar> cycles through all available elements and brings these to the foreground.
With the curly brackets '{' and ''}', data points on these lines can be made smaller and larger respectively. The round brackets, '(' and ')', do the same but are reserved for uniquely for the measured curve. See keyboard functions for more details.

The "cursor" can be moved with the <left> and <right> keys, combined with several combinations of <Shift>, <Ctrl> and <Alt> for larger steps. The base step is the same configurable step size for profile shifting ("<", ">").
Files can also be opened by drag-and-drop into the graph area. Multiple files can be dropped or opened when the history list is activated.
