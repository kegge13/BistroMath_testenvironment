Data validation

If you want to help with data validation then pleas collect the folowing data. Describe the corner defines the negative values for inline and crossline positions.
  • Inline profile with wedge field with toe (high values, thin side) towards G. Specify the collimator angle.
  • Crossline wedge profile with toe of wedge towards A. Also specify the collimator angle. 
  • A pdd.
    To be sure also rotate the tank by 90 degrees and repeat the measurements above after correct setup of the measuring software. Alternatively you might only virtually rotate the tank in the software. Turn the collimator in the same direction with the scans above.
