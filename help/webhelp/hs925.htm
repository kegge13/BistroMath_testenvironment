Known limitations

18/09/2020
Graphic window zoom does not respond keyboard beyond initial zoom value
When using <arrow-down> in zoomed state repeatedly the graphics will not respond beyond the initial minimal zoom range.
It is adviced to configure the ZoomRange with a maximum of 1.04.
There are two work-arounds:
  • Use the mouse to draw a rectangle from uppr-left to lower-right of intended range.
  • Use <Z> twice.
