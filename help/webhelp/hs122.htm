Resampling data
measurement menu 
Many application cannot handle irregular sampled data. With Resample data in the Measurement options this can be corrected.
Resampling is done by applying a local fit to the data and thus, by its very nature, involves filtering. It is controlled by two values in the Settings tab:
- Resample grid: The distance between the resampled points. A zero value will use the average sampling distance.
- Calculation width: the region that is used to apply the local fit. When there are less then two points in this region, the two adjacent points will be used for interpolation.
