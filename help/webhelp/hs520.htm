Why I do not see the division of measurement and reference?

If the blue curve is visible, then check at the title what data are displayed.
  • Check that the Reference option "Calculation is..." is set to "Divide by Reference".
  • Check that the Reference option "Automatic load of Reference" is activated.
  • The configured path for the reference files does not contain the right file.
  • The filed parameters of the measurement differ from the reference: You can make a copy of the reference file, give it the correct name and adjust the parameters.
  • The name of linac is different for some reason: Use the Alias tab to correct this.
