Align reference with measurement
reference menu 
If this item is checked the reference is shifted towards the measurement, based on the difference in center position. If no penumbra is found for both measuement and reference the center position is estimated from the derivative. Note that when a division of measurement and reference is generated, both curves are filtered first and then again aligned. Both shifts are controlled with this item.
As fallback there is also a matching method included.
