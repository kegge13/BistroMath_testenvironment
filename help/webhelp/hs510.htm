Why I do not see any data?

Most reasons are displayed in the message bar below the results area.
  • The most obvious reason is that the data are not understood by BistroMath. The data type might be unsupported or a new variant of a supported data format is found. The author is very keen on tackling these situations.
  • There are no data at all in the editor: Is the File option "Ignore Clipboard" activated?
  • Bi-directional or zero steps in scan at point n: The Measurement option "Accept zero steps" is not activated. Of course zero steps or bidirectional steps should not be produced by any decent system.
  • The "penumbra is too wide". Activate the Measurement option "Accept bad penumbra" or increase the limit.
  • The file has a comma as decimal separator (see below). Set automatic detection on.
