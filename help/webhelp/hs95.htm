Files tab
files tab

The Files tab offers a scan of any directory for registered file types, based on the known file extensions, presenting the most essential data of any file. When a multi-scan file is selected, its contents will be presented as a virtual directory.

Opening a file or scan
A file or scan in a multi-scan file can be opened by double-clicking or right-clicking and then selecting "Open".

Deleting a file
A file can be deleted with right-clicking and then selecting "Delete". Deleting a scan from a multi-scan file is not supported.

Changing the reference directory
In the example above first with the radio button "References" the reference directory C:\Documents and Settings\All Users\Application Data\BistroMath\References was selected.and then the directory C:\Documents and Settings\All Users\Application Data\BistroMath was chosen in the directory box. This has enabled the buttun "-> set path". Clicking this button will set the reference directory to the chosen path C:\Documents and Settings\All Users\Application Data\BistroMath which is also displayed at the bottom in the next image. This can be made permanent by saving a configuration in the options menu.

Alternative axis mapping
When all files have a non-standard data representation axis mapping may be used. With a mixed set of standard/non-standard files there is no optimal choice. This option is probably never needed but introduced here to comply with the internal standards of BistroMath.

File sorting
When the file scan is finished, the list will be sorted on the file name. Any column header can be double-clicked to change the sorting order to that column.
