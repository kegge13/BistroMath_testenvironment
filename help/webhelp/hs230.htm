Dose Conversion and Background tab
dose conversion tab 

In this tab configured Film to Dose settings are selected and can be activated per energy. Also the background offset can be set and activade here per energy. The purpose of this tab is to add flexibility to the configuration files.

To work with Film to Dose settings:
  • A Film to Dose curve must be defined for tha actual energy.
  • On the tab that energy must be selected.
  • The name of appropriate the film type must be selected.
  • The Dose conversion must be activated (checked) in this tab.
  • The general Measurement menu option OD to Dose conversion must be activated.
    Now Film to dose conversion will be applied for each activated and correctly configured energy.
    The example below shows where this can be found in the configuration file.

    [WellhoferOD2dose]
    X6.00=EDR2|-163.990|611.910|-504.380|343.770|-84.840|1000.000
    X10.00=EDR2|-72.640|197.240|38.760|-45.010|11.200|1000.000
    X18.00=none|0.000|10.000|0.000|0.000|0.000|10.000

    To work with Background subtraction:
  • Add a line in this tab and select the correct energy.
  • Set a background level.
  • Activate Background correction for selected energies
  • The general Measurement menu option Background correction must be activated.
