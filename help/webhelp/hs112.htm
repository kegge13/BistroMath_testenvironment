Measurement Norm Adjust mode
The primary function of BistroMath is the comparison of relative data. Both measurement and reference are scaled to the same value at some point. For a division this scaling is not critical and made convenient by applying the same scaling point. For gamma analysis the scaling has a major effect on the results. Therefore an additional normalisation factor can be introduced.

Relevancy
Any renormalisation is only meaningful when a comparison with a reference is done.

When not activated
BistroMath will follow the basic rules. No effects are visible. The renormalisation is always reset to 100.

When activated
  • It is only effectively activated in Advanced mode
  • A renormalisation input field is visible after a file is opened.
  • The "cursor" position is set to the absolute normalisation position.
  • When you move the "cursor" (not the mouse cursor) off the the absolute normalisation position the input field will disappear.
  • The renormalisation value will be reset to 100 every time a measurement set is read.
  • In the graph display the measurement data and all directly related data are multiplied with the renormalisation factor.
  • The division result is scaled accordingly.
  • The measurment data get the same multiplication factor as input to the Gamma analysis.
  • When the data are set as Temporary Reference, the measurement data are multiplied with the renormalisation factor. The factor itself is reset.
