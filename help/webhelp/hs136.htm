Gamma analysis
settings tab 




With gamma analysis the differences between a measurement and its reference can be condensed to one number which combines dose errors, typically in the In-Field area and position errors in the penumbra region on a normalised vector base. This vector gives the smallest distance of combined dosimetric error and position error. The unit length is defined as a reference distance in mm (Distance norm) and dose difference in percent (Dose norm). A dose difference can be calculated in two ways: as percentage of the local dose or as percentage of the normalisation value of the profile. In an area around each measurement point the smallest vector is searched. Very fine steps (Search Steps/mm) with interpolation between points of the reference curve are needed to find a meaningful gamma value. Since the calculation is sensitive for noise a median filter is applied afterwards. The analysis of profiles is only done for relative values above a given threshold (Cutoff Percentage) and beyond the buildup region of depth dose curves (Cutoff Depth). Also, the search is stopped when no better point is found within the region limited to the initial distance to agreement mulitiplied by the Search factor. The result can be shown as calculated curve or histogram. The Confidence Limit, defined as the sum of the mean value of the gamma curve and the standard deviation multiplied with 1.5 of the same curve.
The vertical range of the Gamma curve in the graphical display is set with the Histogram range in the Advanced settings tab. Note that changing any settings will make the Histogram tab temporarily invisible to force recalculation in the Analysis tab.
Any renormalisation will change the result markedly.



Limit to 'In-Field area'
When checked the analysis will be limited to the In-Field area. This area is also defined for FFF profiles although it is not flat. This setting is available for both classes of profiles.
Note that this setting is only relevant for the graph panel and the histogram. The confidence limit in the standard rule setup for the results panel is always limited to the In-Field area.

Dose norm 
The equivalent dose difference to agreement in percent for gamma=1.

Local dose
When checked the difference between measurement and reference is scaled to the local measurement value, otherwise the difference is scaled to the normalisation point.

Distance norm 
The equivalent distance to agreement in mm for gamma=1.

Search steps per mm
The number of steps on each mm from the calculation point at which the reference dose is calculated.

Cutoff percentage
The lower limit for gamma analysis at which dose points are incorporated in the results.

Depth dose cutoff  (mm)
The lower depth limit in mm for gamma analysis at which dose points are incorporated in the results for depth dose curves.

Search factor
A limiting value at which the search is aborted. The starting value is the dose difference in the evaluation point with a minimum of 1, multiplied with the Gamma Search factor. When lower Gamma values are found the limit decreases subsequently.
