Calculation methods

general information 
General remarks
The Omnipro software allows measuring points at irregular positions. Moreover, the measuring speed an therefore the measurement point density may be changed during a scan. This implicates that nowhere a regular interval may be expected which complicates all calculations severely.

Extracting data from profile
Values like the maximum value or a position at a given level do generally not coincide with one single point in the curve data. Therefore some way of interpolation is needed. A further complication is the noise in the measurements giving need to include more points in the surroundings.
Two different methods are used in Bistromath.

Linear interpolation
This is done on base of two points: the nearest point and the second nearest point. The interpolation between those two points is straightforward. This method is used for matching two curves and merging two curves. In those case the goal is to stay as closely to the raw data as possible.

Quadratic fit
When final analysis results need to be calculated, a best estimate is wanted for a noisy dataset. Therefore a number of points is taken in the neighbourhood of the nearest point. The number of points is derived from the (average) step size and user defined range called the Calculation width.   This closely resembles the quadratic filter, which uses its own calculation range.
