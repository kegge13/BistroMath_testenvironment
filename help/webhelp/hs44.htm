User dose level

settings tab & measurement menu 


The user dose level in the settings is used to calculate the width and penumbra positions of a profile. A value of 0 switches off this behavior. Any value in the range [0;100> is accepted. This value is only used however when the User Dose Level item in the Measurement menu is checked or the "u"-parameter is used in a Results panel rule.
Note that user level can be combined with sigmoid analysis.
