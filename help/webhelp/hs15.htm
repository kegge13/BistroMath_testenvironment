Getting started with FFF
general information 

As there are not yet generally accepted methods for analysing FFF users of Bistromath should careful explore all possibilities and set the right choices in the configuration file. 
  • FFF detection should be activated (Field Types tab).
  • The FFF detection parameters should be set (Settings tab).
  • Edge detection should be activated and whether or not the Sigmoid model should be used (Advanced settings tab).
    Further decisions must be on:
  • How the top should be modelled (Measurement menu).
  • What should be defined as the center of field (origin, top, edge based field center).
  • How the profiles should be normalised (top, origin, center of field)
  • Whether or not a reference should be corrected for symmetry (Reference menu).
    More detailed explanations can be found in the FFF-topic.
    If this topic is shown by BistroMath after a FFF profile is detected, then with closing this help file you acknowledge that you have explored these options and you have set a intitute wide configuration file.
