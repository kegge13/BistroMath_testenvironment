Conversion of (groups of) profiles
conversion tab 

On this page all files in the Source Directory of the selected file type will be written to the Output Directory in the also slected type. When, for whatever reason, the input filename equals the output file name, nothing is 
done. Nonexistent output directories will be made as needed. For the conversion of individual files the Analysis page is more suitable as any file can be read and saved again in any format. For multiple scan files all scans in the file will be converted. Note that this need a well configured file name generation.

Options:
Recursive search: also subdirectories will be searched; the extra path elements will be added to the outpout path.

Same Path as Source: when checked, the output path equals always the input path.

Lowercase name: this will set the file names only to lower case. The directory part of the file path is written as is.

Generate file names: when checked, output file names will be generated according to a given format.
