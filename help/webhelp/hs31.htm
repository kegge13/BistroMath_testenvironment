Conversion File Name Format

File conversion tab 
When selected, output file names will be generated according to a configurable format. There are three format strings: general, photons and electrons. An example is: "{X:SEEFddd}{E:SEEllddd}". 
The following syntax is used.

 Syntax Value   Result
 { | }  none   grouping
 X: | E:  group label  photons | electrons  (optional)
 S  scantype  I   (Inplane, GT)
      C   (Crossplane, AB)
      D   (PDD)
      B   (Fanline)
      F   (Freescan)
      P   (Plane)
 EE  Energy [MV/MeV] 2 digits
 eee  Energy [kV/keV] 3 digits
 M  Modality  X | E | O
 DD  Profile depth [cm] 2 digits
 ddd  Profile depth [mm] 3 digits
 LL|XX|YY Field Size [cm]  2 digits
 ll|xx|yy  Field Size [mm]  3 digits
 I  Detector  (free text)
 F  Field type  W | O
