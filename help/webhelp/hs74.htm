Synthetic (virtual) profiles
processing menu 
IPEM topical report 1: guidance on implementing flattening filter free (FFF) radiotherapy
In 2016 a comprehensive British report with guidances for FFF beams was published [G. Budgell et al: IPEM topical report 1: guidance on implementing flattening filter free (FFF) radiotherapy; Phys. Med. Biol. 61 (2016) 8360-8394]. One of the methods they propose, escpecially for field size evaluation is the the division of a small(er) field with a large field profile. The resultant profile assembles closely a conventional profile. In the report this is called a virtual profile.
An alternative method is Edge detection with a sigmoid model.



Implementation
Although this is just the already available division of two data sets, measurement and reference, there are some differences.
  • The calculated output is in terms of Bistromath a real profile, not a relative curve.
  • All standard analysis results are derived from the calculated, synthetic profile. Therefore nearly all measurment results get an exclamation sign as annotation.
  • Where, for an standard comparison all parameters should be equal, now the field size and diagonal detection are ignored.
  • The field size of the divisior (reference) should be larger then the divident (measurement).
  • The calculation area is limited on each side to the minimum of the edge of the divident profile + 2 cm and the edge of the divisor profile - 2 cm.
  • Symmetry correction of the divisor (reference) is disabled, except when it is also applied to the measurement.
  • Automatic load of references is disabled, because it contradicts with using multiple data files.
  • The image is unzoomed.
  • This feature is a processing menu function that can be activated at any time.
  • The graphical analysis is done such that the shoulder match with the profile of the large field.

    Limitations
    There are no known special limitations other than already in place for curve comparison.

    Testing it yourself
  • Load selftest32_x7c40.txt.
  • Set this profile as Temporary reference (<Ctrl+T>).
  • Load selftest32_x7c15.txt.
  • Activate Synthetic profile in the Processing menu (<Ctrl+F>).
