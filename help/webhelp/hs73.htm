Mirroring a measurement to Reference
processing menu 

Logics
When checked, the current profile is copied to the background reference storage. Then this copied profile is mirrored around the Center of Field. Now the Temporary reference item is activated and the new reference loaded as any other temporary reference. The state of this item will not change until switched off again and will therefore work on any next profile. Taking any other temporary reference or clearing the temporary reference will also unset this option.

Implementation
  • First the measurement is copied to the reference.
  • Any applied shift to the data is undone.
  • The double value of any given rotation point is added to the original shift value.
  • Then all position values are inverted, effectively mirroring around the origin.
  • The combined shift as defined above is applied, effectively mirroring around the given rotation point.
  • The data are re-ordered to a increasing position value.
  • The displayed name of the reference is changed to "composite(original name and date)"

    Postprocessing
    The new reference is handled as any reference according to the choices made in the Reference menu. The alignment to the measurement needs special attention here.
    If the reference is not to be aligned nothing is done to the position of the reference. Otherwise the chosen Center of Field definition for the measurement becomes important. The differerence in CoF between measurement en reference is applied to the reference.
  • Origin as CoF (all profile types): the reference will not be shifted.
  • Field edge as CoF (all profile types): basically the 50% points are used to determine the CoF. If a fallback strategy is set in the Advanced settings Tab this may be replaced by some "highest gradient" method.
  • Peak as CoF (FFF profiles only): the top position as derived from the Top model or the Slopes model is used.
  • The maximum is used as CoF (all profiles): this may have an unpredictable effect for most "real live" data.

    Review for FFF fields
    When either the Origin or Field edges are used as CoF a solid comparison is possible. Because a deviation from the Top position from the origin (if coincident with the collimator rotation point) is strongly related to asymmetry this choice amplifies the difference in symmetry between both profiles.
