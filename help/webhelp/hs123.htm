Shifting profiles
keyboard functions


With the keys "<" and ">"  (on most keyboards <Shift>+<,> and <Shift>+<.>), the profile is manually shifted to the left or right in configurable steps. Although the analysis is repeated, the reference profile will not be shifted. It stays in the position decided at read time. The Move profile to Origin in the Measurement menu first moves the profile center to the origin and then matches a reference to this profile.
When files are saved from the main window, any shift will be incorporated in the data. The editor window always contains the original, unshifted data and has its own file write options.
