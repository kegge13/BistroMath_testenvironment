Field Types
general description 
Introduction
During analysis are marked with a Field type:
  • Standard all profiles not assigned to another type and all depth dose curves,
  • FFF  profiles with proven FFF characteristics,
  • Small  profiles with a field width smaller than the given limit,
  • Wedge profiles with proven wedge characteristics.
    Bistromath makes no assumptions on the size and position of the profile in relation to the origin, nor the position of the normalisation point. This introduces the need for a dual loop analysis in which first provisional, on the maximum based, borders are evaluated to set a provisional center and or normalisation point. In the second loop all these values are finalised.

    Field type assigment rules
    With the exception of explicite wedge angle information are assigned to a non-standard type only when proven against the set rules.

    FFF
  • FFF detection must be activated.
  • The dose difference at both ends of the analysis area with the center value should be at at least nn%, with a default value of 10%
  • The difference between 50% positions and derivative edge positions should be at least xx mm, with a default value of 1 mm.
    Both nn and xx can be set in the FFF detection and analysis box on the Settings tab. See FFF profiles for more details.

    MRlinac
    There are no fail prove methods to distinguish a MRlinac profile in the data. Therefore a mapping is available to link treatment unit names to this field type.

    Small field
  • Small field detection must be activated.
  • The field width should be smaller than n cm, with a default value of 2.5 cm.
    This value can be set in the Edge detection box on the Field Types tab.

    Wedge field
    At least one of the criteria should be met:
  • The given wedge angle is non-zero.
  • f * Left 90%-position is larger than the field center or f * Right 90%-position is smaller than the field center.
    The factor f has a default value of 1 and can be set in the Edge detection box on the Field Types tab.
