Quick Start guide

This guide tries to describe different application scenarios and is work in progress. All comments are highly welcomed.

Before you start
In most cases Bistromath will be used in combination with some scanning software. It is advised to configure the presentation of profiles (orientation and sign) on the screen such that it corresponds with the representation of the scanning software. Notably for inplane profiles there is a large variation between vendors. There are also choices to make how depth doses should be normalised for calculations and presentation relative differences with a reference curve. As default Bistromath calculates a division of measurement and reference profile. But this might also be a gamma analysis.
If the the choices above are based on an institution wide policy, they should be stored in the locked file institute.ini.

Adjusting beams
Although absolute criteria are certainly important, the essential feature of BistroMath is the detailed comparison with some reference situation. Bistromath offers you both in one go. 
The process starts with choosing a reference scan. Open this file and store it as temporary reference. If the reference is of another machine you should ignore this to make the reference acceptable. Now each measurement of the same type, depth and field size will be readily compared with the reference.

QA of linacs
In the opinion of the author of Bistromath QA-measurements should always be compared with a predefined reference. Also this is the unique feature, not offered by any major vendor. See the reference overview topic for all details.
