Settings tab

settings tab 


General settings can be found here.

Filter width
Noise reduction is archived by application of a moving second degree polynomial function. The filtering witdth can be set through this option (see Calculation details ). The width in mm can be set for all filtering actions with a default value of 6 mm.


Calculation width
The width in mm can be set for the used number of points for normalisation. At least two points are used, to accommodate for interpolation. This width is also used for the calculation of dose levels (d90, d80, d50, d20). See also the notes on extracting data.


Resample grid
When a fixed data point spacing is needed, this can be set here. It is only used when the Resample data item in the Measurement menu is checked.
The spacing is set in mm; a value of zero is interpreted as resampling with the average sample width. 


Normalisation level
The standard normalisation level may be adjusted to any convenient level. This feature produces results outside the scope of the current conventions in radiotherapy and is neither stored or read in the configuration.


User Dose Level
A dose level defined by the user may be used to calculate the width and penumbra positions of a profile. A value of 0 switches off this behavior.


X / E penumbra h/l levels
The high and low level for the profile penumbra width can be defined here for photons (X) en electrons (E). Defaults are 80% - 20% and 90% - 50% respectively.


Default beam energy [MV / MeV]
When no information on the energy is found in the data, this energy will be assumed.


Histogram range
For a division of measurement and reference the results within the In-Field area are sorted to bins from [-range;+range]. For gamma analysis all available results are sorted in bins from [0;+range].


Assign origin to all 2D array lines
Specific implementation for the StarCheckMaxi 2D-array. The measurement point in the origin is only assigned to the TG line. With this setting it can be inserted into the other lines.
This will be allowed only when the device is in DOSE mode, not in DOSERATE mode (TASK_NAME=DOSE).


Default SSD [mm]
When no information on the SSD is found in the data, this value will be assumed.

Shift Settings: Inplane/Crossplane/Beam-offset
A profile may be given a fixed offset manually for user specific reasons. This value is not stored and is always reset to 0 at program start, but will be applied to every  or depth dose profile. This can be practical for devices which cannot set the origin easily.  Note that these values are introduced in the preprocessing stage immediately after the axis mapping. The old PDD shift was set as a postprocessing shift.

Shift Settings: Shift step
With the keys "<" and ">" the profile or pdd may be shifted to the left or right in steps. The step size can be set here and is stored in the configuration. The same step size is used to move the cursor.


Linac symmetry error
When in the Measurement menu the option Linac symmetry error is checked, the Linac symmetry error is calculated within a ring shaped area of the beam. The inner and outer radius (as defined at 100 cm) can be set here. The result sign can be inverted by checking the relevant Invert sign checkbox.



Merge settings: Profile Shift (cm)
Prior to the actual merging of two profiles, the profile to be merged can be shifted.

Merge settings: PDD Shift (cm)
Prior to the actual merging of two depth does, the lower part can be shifted.

Merge settings: Use matching (cm)
Prior to the actual merging of two profiles, the profile to be merged can be shifted by a matching procedure. The shape of the profile should be sufficient non-flat to work reliably. On the start of the match process the best result from either no shifting or preset shift is the goal to be improved.

Merge settings: Scale overlap area.
Possible differences in normalisation at measurement time can be corrected. The ratio all summed values in the overlap area (after shifting and/or matching) is used to rescale the scan to be merged.


Gamma analysis settings: Dose norm 
The equivalent dose difference to agreement in percent for gamma=1.

Gamma analysis settings: Distance norm 
The equivalent distance to agreement in mm for gamma=1.

Gamma analysis settings: Cutoff Depth (mm)
The lower depth limit in mm for gamma analysis at which dose points are incorporated in the results for depth dose curves.

Gamma analysis settings: Cutoff Percentage
The lower limit for gamma analysis at which dose points are incorporated in the results.

Gamma analysis settings: Search Steps per mm
The number of steps on each mm from the calculation point at which the reference dose is calculated.

Gamma analysis settings: Search factor
A limiting value at which the search is aborted. The starting value is the dose difference in the evaluation point with a minimum of 1, multiplied with the Gamma Search factor. When lower Gamma values are found the limit decreases subsequently.


Mayneord settings: Dmax
Any negative value for Dmax will override the value as found from the beam analysis.

Mayneord settings: SSD org
Any non-zero value for SSD org will override the value for the original SSD as found with the beam data.

Mayneord settings: SSD new
Any non-zero value for SSD new will be taken instead of the Default SSD. This value will be set to the configured default SSD at program startup when no value is found in the configuration file.
