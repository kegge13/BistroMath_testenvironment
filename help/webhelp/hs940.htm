Configuration files

In the configuration topic a complete overview is given.
A limited number of configurable settings can be accessed through the configuration tab:

The most essential configuration for daily use is the reference depth and value for photon beams.
The reference depth is used to normalise depth dose scans. Thus all displayed analysis values are based on this normalisation  ("AbsDepth/AbsValue"). The graphical display of the depth dose curve has its own normalisation ("RelDepth/RelValue"). When combined with a reference file calculated division will be normalised at the same depth. Although the shape of the division will not change drastically the found minimum and maximum will be different. When using generic beams or reference dosimetry at a specific depth this might be the optimal setup.
When the dosimetrical system is based on a depth beyond dose maximum, all depth dose curves should be evaluated realative to this reference point. The text "PDD" is then changed to "RDD".

These values are joined with Dose Conversion parameters. Because there can be different film types this may lead to multiple dose depth definitions for one energy. Starting with version 2.00 all definitions for one energy are kept uniform.



Settings without application interface

BistroMath maintains a configuration file, bistromath.ini / institute.ini, which stores all settings from menu's and windows.
Also some extra data are read from this file.

The ZoomRange sets the scaling for zooming on profiles; a value of 1.03 will show the range from 97% to 103%.
See Special modes for information on the SpecialMode items.

[AnalyseForm]
ZoomRange=1.03
BadPenumbra=3
;the minimum size of data on the clipboard to be accepted as potential legal file format
MinClipBoardBytes=100
;parameters for the Nelder-Mead optimisation engine
;the ENR lower limit for global fit restart as discussed above 
ENR=2.0
;maximum calculation time per start
NMsec=2.0
;maximum number of restarts
NMrestarts=6
;number of significant digits in output
NMdigits=8
;maximum number of cycles,
NMcycles=0


The directory for the reference files (RefDir) may be moved. The filterwidth (Filter) is given in cm. The default oriëntation of the watertank is given in the I(inplane)C(rossline)D(epth) item. The number of  points to calculate a dose in any curve using a second order fit (Dpt) can be increased above the minimum of 3.

[WellhoferObject]
RefDir=C:\ProgramData\BistroMath\references\
Filter=0.60
D20=0.200
D50=0.500
D80=0.800
D90=0.900
ICD=-YX-Z
Dpt=3
multiscan_nr=0
multiscan_max=0
;the Derive function searches for the highest peak and the lowest valley, based on statistical analysis
DeriveMinMax         :twFloatType= 0.9;  {allowed maximum value in first and last point for statistical analysis}
DeriveBinFraction    :twFloatType= 0.25; {maximum of largest bin}
DeriveLookAhead      :Integer    = 3;    {number of points to look ahead to decide on crossing bandlow and bandhigh} 
;parameters for Nelder-Mead fit
ENR=2.0
NMsec=2.0
NMrestarts=6
NMdigits=8
NMcycles=0
multiscan_list=STARCHECKMAXI
