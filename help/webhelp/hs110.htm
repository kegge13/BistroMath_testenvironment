View menu
view menu 


There are a lot of graphical elements. See Graph area for methods for selecting and bringing them to the foreground.

Measurement, Points, Calculated, Reference and Buffer curve
The shortcuts "M", "P", "C", "R" and "B" can be used to selectively show one of these curves.
The Points curve is a series of points, linked to the values as stored in the Measurement curve.

In-Field Area indicators
The shortcut "I" toggles two dotted vertical lines which show the limits of the In-Field area where flatness and symmetry are calculated. See Calculation details.

Penumbra sigmoid indicators
The shortcut "E" toggles the sigmoid model for the penumbra of a profile.
The sigmoid fit data will be taken from the selected curve (if applicable and valid) in a darkened color of the selected curve.

FFF indicators
The shortcut "F" toggles two elements:
  • Left and right slope: the length of these lines represents the calculation area.
  • Top model: the width represents the data range used for this model.

    Values
    The shortcut "V" toggles the display of yellow dots. The normalised value for each shown curve is displayed in the bottom right corner.
    Accelerator keys are (any combinations of) <Ctrl>, <Shift> and <Alt>.

    High resolution values
    When checked an extra decimal is displayed.

    Measurement Norm Adjust mode
    The primary function of BistroMath is the comparison of relative data. Both measurement and reference are scaled to the same value at some point. For a division this scaling is not critical and made convenient by applying the same scaling point. For gamma analysis the scaling has a major effect on the results.     Standard Panel setup
    When this option is activated the default Results panel rules will be added to the rules list, overwriting all rules with identical rule id's. As they are added to the bottom of the list all "surviving" rules will be executed first, but these results will be overwritten mostly by the standard rules. The standard rules are within the code and therefore independent of any configuration file.
    When deactivated all panel rules will be removed. 

    Zoom profiles
    When "Zoom profiles is checked, profiles will be zoomed to the value defined by ZoomRange in the configuration file. The default is 1.03. The <arrow-Up> and <arrow-Down> key may be used to change the ZoomRange. If unzoomed <arrow-Up/Down> are linked to the right axis. Note that the <Z> key is attached to this function.

    Unzoom
    The Unzoom function unzooms the profile. The Zoom key-status is not changed however, thus preserving the autozoom behavior. Note that the U-key is attached to this function.

    Unzoom FFF autmatically
    When set, always Unzoom is done for a profile with proven FFF-features. The underlying criteria are explained here.

    Unzoom PDD autmatically
    When set, always Unzoom is done for a depth dose curve.

    Left Axis lower limit is zero 
    When set values below zero, as found in the first derivative will not be shown.

    Right axis linked to grid

    The horizontal grid lines are based on the left axis. When the right axis is linked to the grid the number of labels is decied by the grid, resulting in "less nice steps" (right part of example above).  When unlinked nice steps can be set freely, mostly resulting in a more quiet view (left part of example above).

    Smart scale electron PDD 
    When checked, the bottom axis maximum will be set around 2*R50.

    Positions in mm
    This will present all lengths and posiitons in mm. For the analysis results in the Results panel this will be done only if the configured multiplier is left at the default value of 1.

    Swapping GT | AB | UD | LR indicators
    The direction of the data representation on the screen may be inverted when comparing side by side with the measuring system. 
    With these swapping options only the axis designations (GT/TG | AB/BA | UD/DU | LR/RL) are swapped. Nothing is changed to the data or analysis results.
    The diagonal scan indicators are handled as LR-type here.
    When swapping is active the bottom axis title is red except when Bottom axis always black is checked.
    The data itself may also be swapped independently at file read time. 

    Bottom axis always black
    When checked axis swapping will have no color effects. 

    No default Edge annotation
    When edge related results in the results panel reflect the primary edge settings, the related annotations can be suppressed.
