Command line parameters
configuration
  • -SpecialMode{number}. See SpecialModes for details.
  • -config any_file. See Institute defaults configuration for details.
  • -references any_directory will use that directory for reference files.
  • -data any_directory will use that path as initial directory for data.
  • -preset any_preset_name. See Configuration for details.
  • -advanced start BistroMath in advanced mode.
