Keyboard functions


All menu items have keyboard shortcuts. Different modifiers are used for each menu

<Ctrl>+<...> File menu


<...> View menu


<Shift+<...> Measurement menu


<Alt+<...> Reference menu


Keys without menu interface
Points size
"(" and ")" respectively decease and increase the display size of the measurement value points. Enlarging automatically activates the visibility of the points, a zero value unsets the visibility.

Line width
When a graphical element is selected, its line width can be changed with "{" and "}". Selection is reported in the message bar.

Line selection in the data plot of the Analysis tab
Use the spacebar to cycle through all selectable elements.

Profile shifting
With the key combinations <Shift>+<,> and <Shift>+<.>, actually being the keys "<" and ">" respectively, the profile is shifted to the left or right in configurable steps.

For files with multiple scans
  • <numpad +>, <PageDown>, <F4>: next scan
  • <numpad ->, <PageUp>, <F3>: : previous scan
    When combined with <Alt+<...> BistroMath will loop through the whole file once automatically (autolooping).
    Note that paging will unfreeze the data.

    Alias list
  • <Ctrl>+<Delete> exit

    During selftest mode
  • <Esc> exit
  • <F5> halt at a failed test (normal state)
  • <F6> step mode
  • <F7> fast mode with halt on failed test
  • <F8> fast continuous mode

    Related to graph area
  • <arrow-Up>/<arrow-Down> change the vertical range in zoomed state or right axis in unzoomed state. For a gamma plot or a histogram the Histogram range is changed.

    Zooming of the vertical scale can be done with the <Up> and <Down> keys. The keys <Z> and <U> set or unset zoomed state. Graphical zooming can be done with mouse left button pressed selecting an area from top-left to bottom-right. The other way around for unzoom.
    The "cursor" can be moved with the <arrow-Left> and <arrow-Right> keys, combined with several combinations of <Shift>, <Ctrl> and <Alt> for larger steps.
