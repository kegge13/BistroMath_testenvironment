Symmetry correction of Measurement / Reference
measurement / reference menu 
If this item is checked a measurement profile will be corrected for asymmetry as far as this can be descibed with the slope of a straight line fitted over the data in the In-Field area (see Calculation details) after this profile is loaded. This is meaningful in the process of adjusting beam parameters tot discriminate between several types of asymmetry.
There are two limits involved here with no access through the user interface but available in the configuration files.

[WellhoferObject]
SymmetryCorrectionLimit=2.00 When the symmetry is above this limit it will not be corrected to avoid extreme results. 
SymmetryCorrectionLevel=0.10 The correction is clipped to the last value when the data fall below this level. 

The method described above does not work well for FFF-profiles. In that case the slope is defined as the sum of the left and right slope of the FFF-profile. For symmetrical profiles the sum will be zero.
For reference profiles with some kind of asymmetry it may be useful to correct this before it is used as divisor on the measurement profile.
When symmetry correctien is applied to the measurement, the flatness and symmetry results are denoted with "(s)".
