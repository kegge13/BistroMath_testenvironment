Presets menu
presets menu 


Preset files contain a limited range of settings, mainly the menu options that can be checked/unchecked. There are a lot of meaningful combinations however that can be related to some "task". As tasks can change a new preset can be loaded. Presets can be made and/or changed by the user. Also, as any ini file, they can be changed with any text editor like Notepad.
The axis swapping options (view menu) and axis sign inversion (measurement menu) are excluded from the presets files when axis remapping is not checked as they are considered as configuration data.
The presets menu is generated from all available presets files. The current preset in use, if any, is checked.
Presets can be loaded from the command line.

Save Preset as...
This is the only fixed item in this menu. A presets file can be saved with this menu option.
