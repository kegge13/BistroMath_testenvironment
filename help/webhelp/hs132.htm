Automatic load of reference files
reference menu 
There is a list reference measurements that are used to evaluate differences in a measurement. To be as unobtrusive as possible in the QA process the given characteristics of the measurement are used to find a reference.
When checked the relevant reference profile is loaded and presented as the Reference curve.
All reference files are selected by the user the first time and loaded as a primary measurement and then stored as reference with a automatically generated name that reflects the contents.
