Unlock/Lock critical options
file menu, options menu
By default saving of reference files or config files with non-default names as well as preservation of data on export is disabled and reset at program start. Locking and unlocking is done with <Ctrl>+<Alt>+<R>. When unlocked, also this menu item will be visible.
