Aliases tab

aliases tab 


With different versions of files it is possible that some string values may differ which should be the same to select the correct reference file. With this alias list this may be corrected.
This feature may also be used to "rename" a linac and thus use reference curves from that linac. Starting with version 2.10 the alias list is not applied on the linac name itself but only on the file name of the reference file.
New lines can be inserted with the <Insert> button. To remove lines use <Ctrl>+<Delete>. There is also a pop-up menu to support these actions. Don't forget to save the aliases if you want to use them permanently. Actually applied aliases for the current measurement are marked red and bold.
When using aliases to cross-reference alternative normalisation methods might be preferred.
