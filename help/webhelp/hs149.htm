FFF profiles

settings tab, measurement menu 
Introduction
Flattening Filter Free beams are beams without a so-called flattening filter which produces a more or less homegeous ouput across the whole field. Essentially this is only optimal for a certain region of depths and large enough fields. With modern planning systems, the need for such a flattened field is not critical anymore. When the flattening filter is removed the output will typically be tripled resulting in much shorter treatment times. The main application is brain tumors, where small fields and high doses are needed. For wide fields the advantage disappears because the intensity has to be adapted by a beam shaping device like the MLCto get the correct dose at each position.
By its nature the Flatness parameter is meaningless for FFF beams. Also the symmetry calculation will perform different on these profiles. Handling these data needs specific detection and analysis.
There is also discussion on the definition of "center of field". The classical approch is to take the average of both field borders. For FFF beams alternative calculations can be chosen.

Detection
There are a lot of non-flat beams which are not FFF. Still the specific features of FFF-class beams need a special treatment.
Therefore a number of criteria are used which all have to be fullfilled. This behavior can be switched on/off as item the Field Types tab. 
  • The field size must be at least 10 cm.
  • Both penumbra's must be found. Incomplete fields are ignored.
  • The scan should be of the AB, GT or Angle type. A diagonal scan with a shifted measuring device will become a "free scan" when the origin is not crossed within 0.5 cm. Shifted scan parallel to the main axes are acceptable.
  • The dose value at the edge of the "analysis area" (in flattened beams the "In-Field area") should drop at least with a given percentage [Minimal dose difference (%)].
  • Inflection point should differ notacibly from 50% border [Min. edge difference (mm)].
    The latter two criteria are configurable and can be changed in the Field Types tab.

    FFF beam tuning
    There is a strong relation between symmetry and center of field. Therefore it can be extremely practical to compare the profile with a mirrored version of itself in the Processing menu. Note that this can be combined with the measurement options for the Center of Field. There is no clear rule whether or not these should be combined. User discretion and insight is essential here.

    Analysis and used parameters
    There is not yet a general agreement on criteria for FFF beams. For QA comparing with a reference profile and evaluating the relative flatness, the core feature of Bistromath, is by far the best choice. For comparing different FFF beams some standard description is needed. The "roof shaped" area can be fitted with straight lines. The length of these lines represent also the calculation range for the slope. The central dome can be modelled with a second order function, in BistroMath called the Top model.
    Both the maximum of the Top model or the intersection of  the slopes can be used to define the center of the field. Alternatively, the top can be used as alignment criterium without choosing this as center definition.
    The slope area is at the outside based on the definition of the In-Field area (for a flattened beam) and at the inner side on the radius of the central dome shape. An extension value [In-Field area extension (cm)] in the Settings tab can push the outer side further outwards or inwards with negative values.
    The gradient is a measure for the FFF shape. The difference in measure can be a measure for the symmetry.
    Thus the average of the (absolute) slope is presented in percent per cm. The symmetry is calculated conventionally. Symmetry correction is done by applying the average of upward and downward slope over the whole profile.
    The definition of relative flatness remains unchanged.
    The related graphical objects can be hidden through the view menu.
