Local Peak Analysis
measurement menu 
 

After default analysis the cursor is moved to the normalisation position. When there are multiple peaks in a single profile, the cursor can be moved to another peak. With this menu item or the keyboard shortcut Ctrl-Q, a peak search will be limited to levels higher than 50 percent of the local maximum. A local normalisation value will be found to start the peak analysis.
